In this article, you'll learn how to upgrade your PHP version on XAMPP. Here, we'll demonstrate upgrading the PHP version from 7.4 to 8.1. However, you can use this guide for upgrading other PHP versions.

  1. Open your browser, then visit this link: https://www.php.net/downloads

  2. You can click on the "Windows downloads" option to download the PHP binary for windows. Download the Thread Safe version.

  3. Once the download is finished, you can extract the zip file then rename it to php-8.1.

  4. Go to php-8.1 folder, then copy php.ini from php.ini-development.

  5. Open the php.ini file with your favourite code editor. Find the extension_dir (You can use Cmd+F or Ctrl+F). Once it is found, you can enable it by removing the semicolon.

# from:
; extension_dir = "ext"
# to:
extension_dir = "ext"

  1. You can enable the extensions such as curl, gd, mbstring, etc. Don't forget to save the change.

extension=bz2
extension=curl
;extension=ffi
;extension=ftp
extension=fileinfo
extension=gd
;extension=gettext
;extension=gmp
;extension=intl
;extension=imap
;extension=ldap
extension=mbstring
;extension=exif      ; Must be after mbstring as it depends on it
extension=mysqli
;extension=oci8_12c  ; Use with Oracle Database 12c Instant Client
;extension=oci8_19  ; Use with Oracle Database 19 Instant Client
;extension=odbc
extension=openssl
;extension=pdo_firebird
extension=pdo_mysql
;extension=pdo_oci
;extension=pdo_odbc
extension=pdo_pgsql
extension=pdo_sqlite
extension=pgsql
;extension=shmop
extension=sqlite3

  1. Move the php-8.1 folder to C:\xampp (I assume you install it in C drive).

  2. Rename the php folder (default php) to any name, let's say php_old. Then rename the php_8.1 to php.

  3. Open your terminal, then check the PHP version. If you've already put the PHP variable in Environment variables before, you'll see it in the output PHP version 8.1.

  1. Now, you can try to install the most recent Laravel version

laravel new laravel-9

  1. Once the project is created, you can go to your project and start up the PHP built-in server.

cd laravel-9
php artisan serve

  1. If you go to your browser, you'll see Laravel 9 welcome page displayed on the screen.
